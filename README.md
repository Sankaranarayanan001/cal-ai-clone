# Cal AI Clone 🥗

An AI-powered calorie tracker that uses Groq Vision AI to analyze food photos and provide instant nutritional breakdowns.

## Features

- 📷 Camera/photo upload → AI food recognition
- 🔢 Instant macro breakdown (calories, protein, carbs, fat)
- 📔 Daily food diary with running calorie total
- 📊 Weekly history charts
- 🔥 Streak tracking
- 🎯 Custom daily goals

## Tech Stack

- React Native + Expo
- Groq Llama 4 Vision API (AI food recognition)
- Expo File System (local storage)
- React Navigation (tab navigation)

## Setup Instructions

1. Clone the repo:
   git clone https://github.com/Sankaranarayanan001/cal-ai-clone.git
   cd cal-ai-clone

2. Install dependencies:
   npm install --legacy-peer-deps

3. Get a free Groq API key:
   - Go to console.groq.com
   - Sign up free
   - Create an API key

4. Add your API key:
   - Open services/gemini.js
   - Replace 'your_groq_key_here' with your actual key
   - get the API_key from the grok ai

5. Run the app:
   npx expo start

6. Scan QR code with Expo Go app on your phone
